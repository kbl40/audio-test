Audio Integration Test

This is a quick test of integrating audio into a web application. The intent is to eventually include audio into web3 dapps. I've accomplished this using the HTML5 audio tag and am establishing control in Javascript using the addEventListener method.

Controls are simple: click Play Music to loop the audio track. Double click Pause Music to pause the audio track.

This short demo is done in collaboration with HeatleyBros. Check them out on YouTube with the link below.
